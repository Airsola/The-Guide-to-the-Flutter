## flutter_widget_demo 

flutter_widget_demo目录下的 Flutter 工程是手册里介绍的 Widget 的 Demo 代码

#### 使用方法

在VS Code 中，打开本目录下的工程即可。

1. `File` -> `Open`

2. 选中 flutter_widget_demo 目录，然后点击 `Open`
